options

A list of options.

options
Properties
defaultZoomable ((string | Element)) : Zoomable elements by default. It can be a css selector or an element.
enableGrab (boolean) : To be able to grab and drag the image for extra zoom-in.
preloadImage (boolean) : Preload images with attribute "data-original".
transitionDuration (number) : Transition duration in seconds.
transitionTimingFunction (string) : Transition timing function.
bgColor (string) : Overlay background color.
bgOpacity (number) : Overlay background capacity.
scaleBase (number) : The base scale factor for zooming. By default scale to fit the window.
scaleExtra (number) : The extra scale factor when grabbing the image.
scrollThreshold (number) : How much scrolling it takes before closing out.
onOpen (Function) : A callback function that will be called when a target is opened and transition has ended. It will get the target element as the argument.
onClose (Function) : Same as above, except fired when closed.
onRelease (Function) : Same as above, except fired when released.
onBeforeOpen (Function) : A callback function that will be called before open.
onBeforeClose (Function) : A callback function that will be called before close.
onBeforeGrab (Function) : A callback function that will be called before grab.
onBeforeMove (Function) : A callback function that will be called before move.
onBeforeRelease (Function) : A callback function that will be called before release.
Example
// Default options
var options = {
defaultZoomable: 'img[data-action="zoom"]',
enableGrab: true,
preloadImage: true,
transitionDuration: 0.4,
transitionTimingFunction: 'cubic-bezier(0.4, 0, 0, 1)',
bgColor: 'rgb(255, 255, 255)',
bgOpacity: 1,
scaleBase: 1.0,
scaleExtra: 0.5,
scrollThreshold: 40,
onOpen: null,
onClose: null,
onRelease: null,
onBeforeOpen: null,
onBeforeClose: null,
onBeforeGrab: null,
onBeforeMove: null,
onBeforeRelease: null
}
