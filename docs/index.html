▸ grab(x, y, scaleExtra, cb = options.scaleExtra)

Grab the Element currently opened given a position and apply extra zoom-in.

grab(x: number, y: number, scaleExtra: number, cb: Function?): api
Parameters

y (number) The Y-axis of where the press happened.
scaleExtra (number) Extra zoom-in to apply.

Returns

options

A list of options.

options
Example
// Default options
var options = {
  defaultZoomable: 'img[data-action="zoom"]',
  enableGrab: true,
  preloadImage: true,
  transitionDuration: 0.4,
  transitionTimingFunction: 'cubic-bezier(0.4, 0, 0, 1)',
  bgColor: 'rgb(255, 255, 255)',
  bgOpacity: 1,
  scaleBase: 1.0,
  scaleExtra: 0.5,
  scrollThreshold: 40,
  onOpen: null,
  onClose: null,
  onRelease: null,
  onBeforeOpen: null,
  onBeforeClose: null,
  onBeforeGrab: null,
  onBeforeMove: null,
  onBeforeRelease: null
}
